Anmerkungen:

1) Das Sketch-Verzeichnis wird über die Arduino IDE > Voreinstellungen festgelegt
2) Dateien (.ino-Sketches) haben IMMER die selbe Bezeichnung wie ihre Ordner
3) Das Verzeichnis "libraries" ist die allgemeine Bibliothek des Arduino
4) Nach dem kompilieren stehen die Daten via COM3-Port zur Verfügung (d. h. unabhängig vom Speicherort der Sketches)
